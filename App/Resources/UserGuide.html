AutoInput User Guide – v{{APP_VERSION}}

AutoInput automates key or mouse presses on a timer — great for repetitive actions or testing. This guide walks you through the basics in a few minutes.

1. Getting Started

  • Pick your target input — a key (e.g. F) or mouse button.
  • Set the interval between presses (from 0.1s up to 10 minutes).
  • Press Start or the global hotkey (F8 by default).
  • AutoInput runs until stopped or until it reaches your set input count.

2. Hotkey Setup

  • Hotkey — Starts/stops AutoInput globally (even when minimized).
  • Target Key — The key or mouse button AutoInput will press repeatedly.
  • Click the button to rebind — then press any key or mouse button to assign.

3. Scheduling

  • Enable Start At to begin at a specific time.
  • Enable Stop At to end automatically.
  • Perfect for time-based testing or automation overnight.

4. Sequence Mode

  • Create step-by-step sequences — e.g. press 1, wait 0.5s, press 2, wait 1.0s.
  • Each full run of your sequence counts as one input.
  • Switch between regular and sequence mode anytime.

5. Configs

  • Save and load presets using the Config tab.
  • Saved as simple JSON files for easy sharing or backups.

6. Quick Tips

  • Check the Status Bar for total inputs and elapsed time.
  • Reset restores defaults if something feels off.
  • AutoInput minimizes automatically when running — just press the hotkey again to stop.
